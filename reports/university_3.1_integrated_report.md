## 📚 학습 내용 요약

### 핵심 개념

```
Lowering: 고수준 IR → 저수준 IR 단계별 번역
Pass: 각 단계에서 변환/최적화를 담당하는 검사관

예시 흐름:
Linalg (수학적) → Affine (구현) → LLVM (기계)
```

### Pass의 종류

```
1. 변환 Pass: 구조 변경 (Linalg → Affine)
2. 최적화 Pass: 효율화 (Constant Folding)
3. 분석 Pass: 정보 수집 (데이터 흐름 분석)
```

```
"설계도를 한 단계씩 구체화하기"

Step 1: "행렬을 곱해"
        → linalg.matmul

Step 2: "3개 루프로 하나씩"
        → affine.for { affine.for { } }

Step 3: "메모리에서 읽고 CPU로 계산"
        → llvm.load, llvm.mul, llvm.store
```
