## 📚 학습 내용 요약

### 핵심 규칙
```
모든 연산 뒤에는 콜론(:)을 붙이고
재료와 결과의 종류(Type)를 정확히 적어야 한다!

형식: %결과 = 도구 %재료 : 타입
```

### 기본 4가지 타입
```
i32  ← 정수 (Integer, 32비트) - 가장 흔함
i64  ← 큰 정수
f32  ← 실수 (Float, 32비트) - AI 학습용
f64  ← 정밀 실수
```

### 타입 매칭 규칙
```
addi/muli/subi → i32, i64 (정수만!)
addf/mulf/subf → f32, f64 (실수만!)

규칙 위반 = 컴파일 에러!
```

```mlir
// 6️⃣ 여러 연산 조합

```

### 핵심 깨달음

1. **타입은 "안전장치"**
   - 도구와 재료가 안 맞으면 에러
   - 컴파일러가 우리를 보호해줌

2. **초등 1.1+1.2+1.3 = MLIR 기초**
   - %이름표 (결과 담기)
   - Dialect.도구 (도구 선택)
   - :타입 (재료 명시)

3. **타입 없이는 불가능**
   - 모든 연산에 필수
   - 안 쓰면 컴파일 에러
