## 📚 학습 내용 요약

### 핵심 개념

```
Tensor (텐서):
- 수학적 개념
- 불변 (Immutable)
- 추상적 ("어디 저장되어 있나" 모름)
- 고수준 코드용

MemRef (메모리 참조):
- 물리적 위치
- 가변 (Mutable)
- 구체적 ("정확히 어디에 저장" 명시)
- 저수준 코드용
```

### 핵심 과정: Bufferization

```
Tensor → MemRef 변환
= 추상 개념 → 메모리 구조화

이것이 대학원 메모리 최적화 논문의 핵심!
```

```mlir
✓ 메모리에 값을 씀 (메모리 수정, 가변)
✅ PASS - 물리적 저장

// 5️⃣ Tensor 삽입 (새로운 텐서)
%0 = tensor.insert %value into %my_tensor[%i, %j] : tensor<4x4xf32>
✓ 새로운 텐서를 생성 (불변성 유지)
✅ PASS - 논리적 삽입

// 6️⃣ MemRef 루프 처리 (실제 구현)
affine.for %i = 0 to 4 {

    %a = memref.load %A[%i, %j] : memref<4x4xf32>
    memref.store %value, %B[%i, %j] : memref<4x4xf32>
  }
}
✓ 메모리를 실제로 반복 처리
✅ PASS - 실제 구현
```
